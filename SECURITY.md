# Security Policy

## Reporting

Please open a GitHub issue for non-sensitive problems.

For sensitive reports, do not include secrets, tokens, private repository links, or personal data in a public issue. Share only the minimum reproduction details needed to describe the risk.

## Scope

This repository contains a Codex skill and documentation. It does not run a server, collect telemetry, or store user data.

The main risks are prompt-content risks:

- accidentally sharing private file paths,
- leaking logs or screenshots,
- copying secrets into examples,
- delegating sensitive work to tools without review.

Review generated prompts before sharing them outside your local workspace.
