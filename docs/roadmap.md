# Roadmap

## v0.1

- Public skill package.
- Install instructions.
- Core examples.
- Lightweight validation script.

## Possible v0.2

- More real-world before/after cases.
- Optional test fixture suite for prompt outputs.
- Separate launch-packaging skill for README/social/release workflows.
- More examples for product design, debugging, docs, and repo maintenance.

## Non-Goals

- Replace project-specific rules.
- Execute tasks directly.
- Become a generic marketing or open-source launch skill.
- Hide unsafe or high-risk operations inside generated prompts.
